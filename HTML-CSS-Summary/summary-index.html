HTML-CSS-Summary

HTML Summary

HTML은 페이지의 내용,구조를 정의하는 마크업 언어입니다.

  • HTML은 내용을 정의하기 위해서 element를 사용합니다
  • HTML element는 일반적으로 태그와내용으로 이루어져 있습니다
  • img같은 void element도 존재합니다
  • 속성을 사용해서 element를 설정할 수 있습니다

CSS Summary

CSS는 페이지 내용을 스타일링 하는데 사용된다.

  • 스타일은 Selector,Property,Value로 정의된다
  • HTML요소에 직접 style속성을 이용해서 스타일링하는것도 가능하다 하지만 추천되지는 않는다.
  • 코드의 중복을 막기위해서 일반적으로 global style을 사용한다.
  • link태그를 사용해서 외부 스타일 시트를 사용할수 있다
  • CSS를 다루기위해서는 상속,명시도(얼마나 구체적인지에 대한 우선순위),박스모델에 대한 이해가 필요하다.
